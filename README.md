# BlazeRootSignal Messenger

BlazeRootSignal is a lightweight, high-performance messaging application developed as a core component of the **BlazeOS** ecosystem. Built with React and Electron for desktop, and Capacitor for mobile.

## ✨ Features
* **Cross-Platform:** Native-like experience across Windows (.exe) and Android (.apk).
* **BlazeOS Integration:** Optimized for low RAM consumption and debloated environments.
* **Real-time Messaging:** Powered by Firebase (Auth & Firestore) for instant connectivity.
* **Modern UI:** Implemented with Expressive Material design aesthetics.

### 🔍 Deep Dive: Operational UI

#### 🔐 Secure Access Console
The **HellRootSignal** login interface is designed with a focus on simplicity and security. It features:
* **One-Click Authentication:** Quick access via Firebase Auth.
* **Test Accounts:** Pre-configured credentials for seamless testing.
* **Connection Status:** Real-time server status indicator at the bottom.

#### ⚔️ Command & Control Center
The main communication hub is built for high-stakes operational environments:
* **Operational Channels:** Segregated channels for *Intelligence (#istihbarat)*, *Operations (#ops-genel)*, and *General Chat*.
* **Role Hierarchy:** Visual distinction between ranks such as *Komutan*, *Teknisyen*, and *Ajan* in the member list.
* **Rich Interactions:** Integrated system messages (v2 rank system updates), emoji reactions, and real-time status checks.
* **Voice Integration:** Dedicated "Lobi" and "Operasyon Odası" voice channels for real-time tactical communication.
* **Network Stability:** Low-latency monitoring (12ms) visible in the top navigation bar.
